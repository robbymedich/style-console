# my-bun-lib

Allow for easy styling of the console output (supports terminal colors)
for JavaScript and TypeScript libraries.

## Install

npm:
npm install my-bun-lib

bun:
bun add my-bun-lib

deno:
import { foo } from "npm:my-bun-lib";

## Usage

import { foo } from "my-bun-lib";

console.log(foo());
